# dockerfile-cookiecutter

[Cookiecutter](https://www.cookiecutter.io/) template to generate a basic Dockerfile
project.

## Pre-requisites

```sh
pip install pipx
```

## Run

```sh
pipx run cookiecutter gh:andreswebs-public-images/dockerfile-cookiecutter
```
